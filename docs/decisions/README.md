# Architecture Decision Records (ADRs)

ADRs document significant architectural decisions made during the project.
Each record captures the context, decision, and consequences at the time it was made.

## How to Add a New ADR

1. Copy `0001-template.md` to `NNNN-short-title.md`
2. Fill in all sections
3. Add a row to the index above
4. Status options: `Proposed` → `Accepted` | `Rejected` | `Superseded by NNNN`
