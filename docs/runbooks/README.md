Operational guides for running and maintaining ExamChecker.

## Index

| Runbook | Purpose |
|---------|---------|
| [local-dev.md](local-dev.md) | Start all 3 services for local development |

## Adding a Runbook

Create a new Markdown file in this directory and add it to the index above.
Runbooks should be step-by-step and include expected outputs for verification.
